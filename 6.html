Sketches Overview

We created our prototype using our sketches, this will be demonstrated by the video. We initially created created 6 sketches. Login, Sign up, Home page, Profile, Jupiter Hub and Forum. We then realised that we needed more sketches for each click or user navigation, so instead of starting over, we improvised and used the existing sketches but placed cut out paper on-top of the already made sketches. So for example, when the user clicks generate user log, while one might expect to be taken to another page , in our sketches they will view the same page but with the python terminal covered by the log history of the user. We repeated this process multiple times for each relevant user action.
