Below are some of the positives and negatives of both applications previously mentioned - these were the main focus of the team while gathering the user requirements.

Name Platform Positives Negatives

  • Provides the ability to upload a dataset and few the analysis via maps, graphs and tables.
  • Allows the generation of individual logs of usage history for each profile
  • Does not provide a Forum
  • Does not provide access to Jupiter Hub
Network of Innovators Website
  • Provides an effective forum platform
  • Provides creation and management of profiles
  • Provides an effective Search functionality
  • Does not provide access to Jupiter Hub
  • Does not provide access to CKAN
