Project Overview

The purpose of our project is to provide a platform for a variety of users to easily access, analyse and discuss geodata, regardless of their background (background here meaning geographic experts, business people, politicians and so on). The data will be previously uploaded geographical data that users will find useful to manipulate - it could be in relation to their local areas, their places of work, geographical places of interest, places they intend to (or have) visited, and so on. The output file passed back to the user can be in a variety of formats including map data, graphs, charts and so on.



Functional Overview
Our project is to create a website for the UCL Urban Dynamics Lab. There are three main features and functions for the user:
  • The website allows for analysis of uploaded geo-data via Jupiter Terminal (CKAN)
  • The website allows for discussion between users via the implementation of a forum
  • The website forum is fairly developed, to a greater extent than the analysis feature (which requires just the integration of the Jupiter Terminal)
The website will thus allow users to sign up and log in, analyse uploaded data, use the forum feature to discuss this, as well as contribute to existing threads, create new threads, edit their profiles, and connect with other users.
